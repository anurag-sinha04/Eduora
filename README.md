# 🚀 Eduora

Deployed Link: https://anurag-sinha04.github.io/Eduora/
### 🧠 Adaptive AI Learning Platform for Accessible Education

Eduora is an AI-powered educational platform designed to make learning **accessible, personalized, and inclusive** for students with different abilities — including visual, hearing, and cognitive challenges.

---

## 🌟 Problem

Traditional learning platforms follow a **one-size-fits-all approach**, which creates barriers for:

- 👁️ Visually impaired users  
- 🧏 Hearing impaired users  
- 🧠 Students with cognitive difficulties  
- 🌍 Multilingual learners  

Most systems only provide basic captions and fail to adapt how content is delivered.

---

## 💡 Solution

Eduora transforms learning by dynamically adapting content based on the user’s needs.

> 🏆 *“We don’t just make education accessible — we make it adaptable.”*

---

## 🔥 Features

### 🎤 Speech-to-Text (STT)
Converts live speech into readable text.

### 🔊 Text-to-Speech (TTS)
Reads content aloud for visually impaired users.

### 🤟 Sign Language Support
Displays sign-language visuals for hearing impaired users.

### 🧠 Cognitive Mode (AI-Powered)
- Simplifies complex content  
- Converts into bullet points  
- Improves understanding  

### 🌍 Multi-Language Translation
Supports multiple languages for inclusive learning.

### 🎥 Video Conferencing Interface
- Real-time captions  
- Adaptive UI based on mode  

### 🎯 Adaptive Learning Modes

| Mode | Function |
|------|--------|
| 👁️ Visual | Audio-based learning + voice navigation |
| 🧏 Hearing | Text + sign language support |
| 🧠 Cognitive | Simplified structured learning |

---

## 🧠 AI Integration

Eduora uses AI to:
- Simplify complex educational content  
- Generate summaries  
- Improve learning comprehension  

---

## 🖥️ Tech Stack

- **Frontend:** HTML, CSS, JavaScript  
- **AI APIs:** OpenAI / OpenRouter / Gemini  
- **Speech Recognition:** Web Speech API  
- **Text-to-Speech:** Browser TTS  
- **UI:** Responsive dark-themed interface  

---

## ⚙️ How It Works

1. User opens the platform  
2. Voice assistant prompts for learning mode  
3. User selects (via voice or click):
   - Visual  
   - Hearing  
   - Cognitive  
4. System adapts content dynamically  
5. AI processes and delivers personalized learning
